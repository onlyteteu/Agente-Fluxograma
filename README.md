# FlowTalk

Aplicacao SaaS para transformar descricoes em linguagem natural em fluxogramas visuais com apoio de IA.

## Stack atual

- Next.js
- TypeScript
- Tailwind CSS
- App Router

## Como rodar

```bash
npm run dev
```

Abra `http://localhost:3000`.

## Proximo foco do MVP

- Captura de texto do usuario
- Geracao de JSON via IA
- Validacao da estrutura
- Renderizacao bonita do fluxograma
- Edicao simples
